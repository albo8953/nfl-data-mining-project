# nfl-data-mining-project
This project focuses on leveraging previously collected data from the NFL to predict future winners and analyze what factors contribute to winning.  It is headed by Aayush, Alan, and Lucas.  Our current goal, with this GitHub and webpage, is to get a concrete overview of our scope, focus, and specific tasks.
The scripts directory contains a claning script in python that processes our data, reporting missing values, removing them, and then using imputation replacing them.  It also conatians bash scripts using our API key to aquire the data we would like from the time periods we desire. 
The misc directory conatins the output of the python script run and is a report of what was done and what values were missing in our data.
The data directory contains the csv/json files of raw and processed data collected from the API we used.
